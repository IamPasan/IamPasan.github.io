## Goal

Find the password for bandit4, stored on the bandit3 server.

## Recon

To sole this need, 

`ls -a` 
		`-a` Used for list all files and folders including hidden ones and also can use `-la` for get the a all files and folders as a long list which shows some additional valuable details

```shell
ssh bandit3@bandit.labs.overthewire.org -p 2220
# password: password from bandit2 - 3

ls 
	# you can see folder named inhere
~/inhere$ ls -a 

cat ...Hiding-From-You

```



> Output:

```shell
bandit3@bandit:~$ ls
inhere
bandit3@bandit:~$ cd inhere/
bandit3@bandit:~/inhere$ ls -a
.  ..  ...Hiding-From-You
bandit3@bandit:~/inhere$ cat ...Hiding-From-You 
xzTXq1rDJQVVAzdv5cHq1TQytTWufAMq
bandit3@bandit:~/inhere$ 


```


> The `...Hiding-From-You` file  in bandit3's `inhere` directory contains the bandit4 password. 
{: .prompt-info }


## Answer 

Password for bandit3 - 4: `xzTXq1rDJQVVAzdv5cHq1TQytTWufAMq`
