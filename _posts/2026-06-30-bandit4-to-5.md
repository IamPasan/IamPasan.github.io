## Goal

Find the password for bandit5, stored on the bandit4 server.


> The password for the next level is stored in the only `human-readable` file in the **`inhere`** directory. 

## Recon

To sole this need, 

`File` - Used to determine file types. inhere we can use that command  with a wildcard on the filename to find the human readable `ASCII`  only file

`file ./-file0*`


```shell
ssh bandit4@bandit.labs.overthewire.org -p 2220
# password: password from bandit3 - 4

ls 
	# you can see folder named inhere
~/inhere$ ls -la 

file ./-file0*

cat ./-file07

```
