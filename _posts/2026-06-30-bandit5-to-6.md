## Goal

Find the password for bandit6, stored on the bandit5 server.

> The password for the next level is stored in a file somewhere under the **`inhere`** directory and has all of the following properties:

- human-readable
- 1033 bytes in size
- not executable

## Recon

To sole this need, 

`find` - Search for files in a directory hierarchy. 

Here we used `find` command to to find the specific file that match with the given properties to find the password for next level. To do that we need to use some  parameters 

- `-type f` - used to filter regular files
- `! -executable` - For find none executable files 
- `-size 1033c` - for filter files that with 1033 bytes (`c` used for bytes)

`find -size 1033c ! -executable -type f`


```shell
ssh bandit5@bandit.labs.overthewire.org -p 2220
# password: password from bandit4 - 5

ls 
	# you can see folder named inhere
~/inhere$ ls -la 

find -size 1033c ! -executable -type f # Full command

#for easy we can use only find the file by size

find -size 1033c

cat ./maybehere07/.file2

```
